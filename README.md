# Roadmap Engenharia de Qualidade & IA

Mapa interativo com 17 blocos de conhecimento para quem quer evoluir na carreira de QE — do básico ao avançado, passando por automação, DevOps, dados, IA e agentes.

🚀 **Acesse agora:** [roadmap-qe-ia.vercel.app](https://roadmap-qe-ia.vercel.app)

📄 **Artigo completo no Medium:** [Leia aqui](https://fabiokaia25.medium.com/1e8606b10939)

---

## Como aproveitar o roadmap

1. Clique em **Iniciar jornada** na tela inicial
2. Use as setas **↑ ↓** ou os botões na barra inferior para navegar entre os blocos
3. Clique em qualquer ponto da linha do tempo para ir direto a um tópico
4. No final, você encontra o artigo completo no Medium e pode recomeçar a jornada

**No celular:** arraste o painel de conteúdo para baixo para ver o mapa completo.
